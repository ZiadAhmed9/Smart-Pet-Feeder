# Smart-Pet-Feeder
In this project i made a smart pet feeder that allows users to feed their pets remotely. 

Components used:
•	STM32F401CC Black Pill development board
•	IR Sensor
•	SG-90 servo motor
•	Ultrasonic sensor
•	HC-05 Bluetooth module
•	Buzzer, push button, and led

Softwares used:
•	STM32CubeIDE
•	ST-utility link
•	Proteus 8.6
